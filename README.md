Optimization to find weights for different configurations that maximize fit with diffuse scattering data.

To run, install the environment with [pixi](pixi.sh) like so:
```bash
git clone (thisrepo)
cd simulation_timeseries_optim
pixi install
```

Files for running the script can be found [here](http://bl831.als.lbl.gov/~jamesh/pickup/diffUSE_CC_opt_test.tgz).
